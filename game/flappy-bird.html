Flappy Birds 2D Live Session

Project Goals

  • Learn to create performant infinite or endless style games by moving the world instead of the player.

  • Learn to optimize your game by using object pooling to create the illusion of many objects using only a few.

  • Create 'cartoon style' non-realistic 2D physics using Unity's built in 2D Physics system.

  • Presenter brings in only artwork and a font. Building from scratch here.

Sprites

  • Import sprites and fonts

  • Switch sprite mode to multiple if sprite is sprite sheet.

  • Slice sprite into appropriate sprites for the sprite sheet.

  • Drag in birds first sprite. Drag in environment objects (grass, sky(background))(sprites).

  • Order objects in scene with sorting order in sprite renderer. Click add sorting layers
    for environment objects (sky - background) (birds - midground) (ground - foreground))

Bird Physics

  • Add rigidbody2d to bird. Add polygon-collider2d to bird.

  • Add box-collider2d to ground. Set appropriate position to the collider.

Bird Script

  • Add "Bird" script to bird. Create folder for scripts. Open for editting.

  • Check if player is dead or not. "private bool isDead = false"
    In update, check if player is dead. If not, check for player input.

  • Make reference to rigidbody2d. On player input,
    set rb.velocity = Vec2.zero, then rb.addForce(0, upforce=200f)

Bird Collision

  • Add OnCollisionEnter2D function to bird script.

  • On any collision we want to set isDead to true.
    Also set rb.velocity to zero.

Bird Animations

  • Open Animation window (ctrl + 6). Select "Create" with bird selected.

  • Name file "Idle" and save in animation folder. This adds and animator controller and other automated steps.

  • With bird selected, add property, go to sprite property and hit plus button.
    Delete second keyframe since it goes nowhere.

  • Create new clip called "Flap" and save.
    Paste the previous animation properties to this one.
    Grab bird hero 1 and drag into sprites field.

  • Create new clip called "Die" and again paste in first anim properties.
    Grab dead bird sprite and add into sprites field.

Bird Animator

  • Open Animator window. Arrange the three states to how you want.

  • Make trigger parameter called "Flap" which will be set in script.
    Make second trigger parameter called "Die"

  • Right click on Idle and make transition to Die state.
    Turn of HasExitTime, because we want Idle to loop until we Die.
    Add Die trigger to Condition.

  • Make transition from Idle to Flap. Turn off hasIdleTime. Set condition to Flap.

  • Make a transition from Flap to Idle.
    Keep default length so it switches back after 1 second.

  • Return to bird script and add Animator reference.

  • Under AddForce, use anim.SetTrigger("Flap");
    In Collision function, use anim.SetTrigger("Die");

Score/Game Over

  • Add UI Text to Scene. Set Text to "Score: 0"
    Label as "Score Text"
    Set appropriate font size. With hori/vert overflow if needed.

  • Set color to white.
    Center/Middle align, and anchor position with alt to "bottom middle"
    Position offset of y axis

  • Add Shadow, with effect distance of 1.8 on x/y.

  • Add another UI Text. Set Text to "Game Over"
    Label as "Game Over"

  • Anchor position to "upper middle" and offset y-axis
    Set other desired properties

  • Duplicate Score Text and make child of game over text.
    Position under Game Over Text.
    Set Text to "Flap to Restart".
    Disable because we don't want to see until game over.

Game Controller

  • Create empty as "Game Controller" with new GameController script.
    Change start to awake. Add UnityEngine.SceneManagment

  • Create function birdDied(). Make public GameObject gameOverText.
    Set gameOverText to true in birdDied.

  • Make public gameOver bool.
    Set to true in birdDied.

  • Make a public static GameController instance for Singleton purpose.
    In awake, if instance equals null, instance equals this.
    else, if instance does not equal this, destroy the game object.

  • Back in bird script, use GameController.instance.BirdDied() function

  • Select appropriate variables for public vars.

  • In update, if game over equals true, and click input happens...
    Use, SceneManager.LoadScene(SceneManager.LoadActiveScene().buildIndex)

Scrolling Background

  • Create ScrollingObject script and add to ground.
    Add rigidbody2d to the ground for scrolling.
    Set motion to kinematic so it's not effected by gravity, just scripting.

  • In the script, make reference to rigid body.
    In Start, set rb velocity to GameController.instance.scrollSpeed, 0.

  • Add public float scrollSpeed as -1.5f to GameController

  • In update, if GameController.instance.gameOver is true, set rb.vel to zero.

  • Create duplicate ground as Ground2.
    Create empty as Scenery and put ground objects as children.

  • Set Ground2 positions x to the size of the backgrounds.

Repeating Background

  • Add new RepeatingBackground script to the ground objects.

  • Add private box-collider2d as groundCollider
    Add private float as groundHorizontalLength

  • In start, get reference to groundCollider.
    Set groundHorizontalLength to groundCollider.size.x

  • Make a new function, RepositionBackground.
    In it, set a Vec2 groundOffset equal to new Vec2(groundHoriLength * 2f, 0)
    Then set the transform.position to (Vec2)transform.position * groundOffset

  • In Update, check if(transform.position.x -groundHoriLength)
    If so, call RepositionBackground

Columns

  • Drag in column sprite. Set Sorting Layer to midground.
    Position Y to -4.

  • Add box-collider2d to column and set appropriate size.

  • Duplicate for a second column and rotate upside down.

  • Create Empty container name Columns to set as parent for columns.

  • Add box-collider2d as trigger to Columns holder object.
    Move slightly past the columns and in between. Cover Gap.

  • Add rigidbody2d to Columns object. Add new Column script to Columns object. Add scrollingObject script to Columns object. Make Columns into a prefab.

Bird Score

  • In GameController, make a private int score and set to 0.

  • Add using UnityEngine.UI
    Declare a public Text scoreText

  • Make BirdScored function, and add a check if game is over. If so, return out.
    Else, score ++. And set scoreText.text equal to "Score " + score.toString()

  • In Column script, make a private void OnTriggerEnter2D(Collider2D other)
    If other.GetComponent() does not equal null, then call GameController.instance.BirdScore()

Object Pool

  • Make a new Column Pool script and attach to Game Controller object.
    System.Collections.Generic to script.

  • Make a public int columnPoolSize as 5.
    Make a public GameObject columnPrefab.
    Make a public float spawnRate as 4f
    Make a public float columnMin as -2f
    Make a public float columnMax as 2f
    Make a private GameObject[] array called columns.
    Make a private Vec2 objectPoolPosition as new Vec2(-15f, -25f)
    Make a private float called timeSinceLastSpawned.
    Make a private float spawnXPosition as 10f
    Make a private int currentColumn as 0

  • In start, set columns to new GameObject[columnPoolSize]

  • In start, create a for loop with i=0, i < columnPoolSize. Set columns[i] = (GameObject)Instantiate(columnPrefab, objectPoolPosition, Quat.identity) as GameObject

  • In update, set timeSinceLastSpawned += Time.deltaTime
    If GameController.instance.gameOver is false AND
    if timeSinceLastSpawned >= spawnRate...
    set timeSinceLastSpawned to 0
    Also make float spawnYPosition as Random.Range(columnMin, columnMax)
    Set columns[currentColumn].transform.position = new Vec2(spawnXPosition, spawnYPosition)
    Add 1 to currentColumn
    Another if saying if(currentColumn >= columnPoolSize)...
    set currentColumn to 0

  • Add appropriate variables to script in scene
