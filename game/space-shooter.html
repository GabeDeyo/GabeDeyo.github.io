Setup

  • Create a new "Space Shooter" project and import SpaceShooter assets.

  • Change build target to Web Player.
    In Project Settings - Player, change default width/height to 600/900

  • Change Game View to 600/900.
    Change layout so that Scene and Game Windows take up full vertical space somewhere.

Player

  • In Models, drag the vehicle_player_ship into the scene as "Player".
    Reset position.

  • Add a rigidbody to the player and deselect "Use Gravity"

  • Add a Mesh Collider to the player.
    Add the imported mesh collider to the player.
    Make this a Trigger Collider

  • Add the Engine prefab to the player.

Camera & Lighting

  • Reset the Main Camera.
    Rotate to (90, 0, 0) and position to (0, 10, 5)
    Change Projection to Orthographic and set Orthographic Size to 10

  • Change background color to black.
    In render settings, change ambient light to black.

  • Create a Directional Light as "Main Light"
    Reset position and rotation.
    Manually rotate towards (20-ish, -115-ish)
    Increase intensity to 0.75

  • Duplicate the Main Light as "Fill Light" and reset completely.
    Rotate to (5, 125, 0)
    Reduce intensity to 0.5
    Change color to (128, 192, 192)

  • Duplicate Main Light again as "Rim Light" and reset completely.
    Change color to white.
    Rotate to (-15, 65, 0)
    Drop Intensity to 0.25

  • Create an empty "Lighting" object and reset.
    Drop all the lights into this object.
    Move Lighting to (0, 100, 0) to get it out of the way.

                rigidbody.velocity = movement * speed;

Spawn Waves

  • GameController 

                public GameObject hazard;
            public Vector3 spawnValues;
            public int hazardCount;
            public float spawnWait;
            public float startWait;
            public float waveWait;

  • GameController 

                void Start() {
                StartCoroutine(SpawnWaves());
            }

  • GameController 

                IEnumerator SpawnWaves() {
                yield return new WaitForSeconds(startWait);

                while(true) {
                    for(int i = 0; i < hazardCount; i++) {
                        Vector3 spawnPosition = new Vector3(Random.Range(-spawnValues.x, spawnValues.x), spawnValues.y, spawnValues.z);
                        Quaternion spawnRotation = Quaternion.identity;
                        Instantiate(hazard, spawnPosition, spawnRotation);

                        yield return new WaitForSeconds(spawnWait);
                    }
                    yield return new WaitForSeconds(waveWait);
                }
            }

  • Set hazard count to 10
    Spawn wait to 0.5
    Start wait to 1
    Wave wait to 4

    Add to Display Text.

  • Create a GUI Text as GameOverText and position at (0.5, 0.6, 0)
    Add to Display Text.

  • GameController 

                public GUIText restartText;
            public GUIText gameOverText;
            private bool gameOver;
            private bool restart;

            void Start() {
                gameOver = false;
                restart = false;
                restartText.text = "";
                gameOverText.text = "";
            }

            public void GameOver(){
                gameOverText.text = "Game Over!";
                gameOver = true;
            }

            //In SpawnWaves, after yield return new WaitForSeconds(waveWait)
            if(gameOver) {
                restartText.text = "Press 'R' to Restart".
                restart = true;
                break;
            }

            void Update() {
                if(restart) {
                    if(Input.GetKeyDown(KeyCodeR)) {
                        Application.LoadLevel(Application.loadedLevel);
                    }
                }
            }
    Apply appropriate objects to their respective fields.
