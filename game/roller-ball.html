Game Setup

  • Create new 3D "Roller Ball" project. Use 3D default project settings.

  • Create new "Mini Game" scene.

  • Create a Plane object and name "Ground". Place at (0, 0, 0).
    In Gizmos, deselect Show Grid option.
    Rescale terrain to (2, 1, 2)?

  • Create Sphere object and place at (0, 0.5, 0)

  • Create Materials folder and create a Material as Background
    Change Albedo to a nice dark blue. (0, 32, 64)
    Apply material to Plane object.

  • Rotate the default light to give the player a better shape.

Move the Ball

  • Create Scripts folder and add "PlayerController" script to the Player

  • Add rigidbody to the Player

  • PlayerController
    private Rigidbody rb;
    public float speed;

  • PlayerController
    void Start() {
      rb = GetComponenet();
    }

  • PlayerController
    void FixedUpdate() {
      float moveHorizontal = Input.GetAxis("Horizontal");
      float moveVertical = Input.GetAxis("Vertical");
      Vector3 movement = new Vector3(moveHorizontal, 0f, moveVertical);
      rb.AddForce(movement * speed);
    }

Move the Camera

  • Lift camera to (0, 10, -10), and rotate to (45, 0, 0)

  • Make a new "CameraController" script and add to camera.

  • CameraController
    public GameObject player;
    private Vector3 offset;

  • CameraController
    void Start(){
      offset = transform.position - player.transform.position;
    }

  • CameraController
    void LateUpdate(){
      transform.position = player.transform.position + offset;
    }

Setting up Play Area

  • Create new empty "Walls" object.

  • Create new cube "WestWall" as a child of Walls.
    Change scale to (0.5, 2, 20.5), push into place.
    Repeat for other walls.

Creating Collectable Objects

  • Create a new cube as "Pick Up".
    Move to (x, 0.5, z)
    Rotate to (45, 45, 45)
    Scale to (0.5, 0.5, 0.5)

  • Create a new script called "Rotator" and attach to Pick Up.

  • Rotator
    void Update(){
      transform.Rotate(new Vector3(15, 30, 45) * Time.deltaTime);
    }

  • Make Pick Up a Prefab asset in a new Resources folder.

  • Make a new empty Pick Ups object.
    Make Pick Up object a child of Pick Ups
    Duplicate and spread Pick Ups around.

  • Make a new Pick Up material and apply to the Pick Up prefab.
    Make it yellow or gold or some shit.

Collecting Pick Ups

  • Change Pick Ups to have trigger colliders

  • PlayerController
    void OnTriggerEnter(Collider other){
      if(other.gameObject.CompareTag("Pick Up")){
       other.gameObject.SetActive(false);
     }
    }

  • Set the tag of the Pick Up prefab to "Pick Up"

Score & Text

  • Add a Text component as "Count Text".
    Change color to white.
    Set placeholder text as "Count Text"

  • Set anchor, pivot, and position to the upper left corner.
    Set position to (10, -10, 0)
    Associate this text with PlayerController countText.

  • Add a Text component as "Win Text"
    Change to white.
    Change size to 24.
    Change alignment to center/middle
    Set text to Win Text
    Change position to (0, 75, 0);

  • PlayerController
    public Text countText;
    public Text winText;
    private int count;

  • PlayerController
    void Start(){
      count = 0;
      SetCountText();
      winText = "";
    }

  • PlayerController
    void OnTriggerEnter(Collider other){
      count = count + 1;
      SetCountText();
    }

  • PlayerController
    void SetCountText(){
      counText = "Count: " + count.ToString()
      if(count >= 12){
       winText.text = "You Win!";
     }
    }
