Credit Card Mask

  • Usually when you buy something you're asked whether your credit card number, phone number or answer to your most secret question is still correct. However, since someone could look over your shoulder, you don't want that shown on your screen. Instead, we mask it. Task: Write a function maskify, which changes all but the last four characters into '#'.

  • Examples 

                Maskify('1234567890123'); // returns '#########0123'
            Maskify('12345'); //returns '#2345'
            Maskify('1'); //returns '1'

  • Best Solution 

                public static string Maskify(string cc) {
                int len = cc.Length;
                if(len <= 4)
                    return cc;

                return cc.Substring(len-4).PadLeft(len, '#');
            }
