Introduction

  • JSON - JavaScript Object Notation

    • Format for sharing data
    • Derived from JavaScript
    • Language independent
    • An alternative to XML

  • Advantages

    • Easy to Read
    • Easy to Parse
    • Leaner than XML
    • Growing support in APIs
    • Natural format for JavaScript
    • Implementation in many languages

JS Objects vs. JSON

  • JSON Strings

    • JSON keys wrapped in "quotes"
    • JSON keys can be any valid string
    • JSON values have to be one of six data types:
      • strings, numbers, objects, arrays, booleans or null

                { "full_name" : "Gabe Deyo" }

  • JavaScript Object

    • Quotes on keys not necessary
    • JavaScript Object can be any data type (including function)

                var info = {
                full_name : "Gabe Deyo"
                getName : function (){ 
                    alert(this.full_name);
                }
            };

  • Parsing JSON

    • JSON has to be parsed into JavaScript
    • Can be done using eval or JSON.parse(). JSON.parse is recommended as it is more secure.
    • JSON.stringify does the opposite of parse
    • A library like jQuery can be helpful when facing compatability issues

                var data = '{ 
                "name" : "Gabe Deyo",
                "position" : "Web Developer",
                "skills" : ["cooking", "cleaning", "others"]
            }';
            var info = JSON.parse(data);
            var infostring = JSON.stringify(info);

Creating Data

  • Keys & Values

    • Multiple values separated with commas
    • Whitespace for clarity
    • No quotes on numbers, true, false, null

                var info = {
                "full_name"  : "Gabe Deyo",
                "title"      : "Web Dev",
                "company"    : "IBI Data",
                "years"      : 1.3
            }

  • Retrieving Values

    • Normally accessed through dot notation
    • Can also be accessed through [] notation

                var info = { "full_name" : "Gabe Deyo" };

            console.log(info.full_name); //Gabe Deyo
            console.log(info["full_name"]) //Gabe Deyo

  • Arrays

    • Lists created with brackets
    • Accessed via an index number

                var info = {
                myArray = [
                "0",
                "1",
                "2",
                "3" ]
            };

            console.log(info.myArray[1]); //1

  • Objects

                var info = {
                "links" :  {
                    "blog"     : "http://1.com",
                    "facebook" : "http://2.com",
                    "youtube"  : "http://3.com",
                }
            }
