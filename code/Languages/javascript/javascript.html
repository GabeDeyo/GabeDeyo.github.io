Introduction

  • • JavaScript is a scripting language that runs in the browser and interacts with the HTML Markup and CSS Rules to change what you see and do.
    • When you point your browser at an address with an html document, it first pulls down the HTML and indexes its content. Then downloads any CSS and JavaScript document referenced, runs the JavaScript to see if any of the HTML Markup has changed, applies the CSS to make everything look the way it's supposed to, and finally leaves the Javascript running in the background, in case an interaction or event triggers further scripts.

  • • JavaScript conforms to the ECMAScript standard, a specification for interpreting code and languages like JavaScript.

Modifying and Accessing Objects

  • Modifying Arrays in Objects

    Deleting an object key/value is easy enough: delete(info.first_item)

    Deleting an index of an array in an object will have an issue though, as it would normally.
      • Using delete(info.links[2]) will actually delete the item, however the array will be broken as it won't actually remove that index.
      • The solution here is to use info.links.splice(2,2) to succesfully remove that index from the array.

    Use pop and push to append and depend items from the array.

  • Looping an Object

                var info = {
                "full_name" : "Gabe Deyo",
                "title"     : "Web Developer",
                "links"     : {
                    "blog"     : "http://gabedeyo.com",
                    "facebook" : "http://facebook.com",
                    "google"   : "http://google.com"
                }
            };

            var output = "";

            for(key in info.links) {
                if(info.links.hasOwnProperty(key)) {
                    output += 
                    '<li>' + 
                        '<a href=""' + info.links[key] +'">' + key + '</a>' + 
                    '</li>';
                } //if the links has the key property
            } // for...go through each link

            var update = document.getElementById('links');
            update.innerHTML = output;

  • Accessing Objects in Arrays

                <body>
                <h2>Links</h2>
                <ol id="links"></ol>
            </body>
            
                var info = {
                "full_name" : "Gabe Deyo",
                "title"     : "Web Developer",
                "links"     : [
                    { "blog"     : "http://gabedeyo.com" },
                    { "facebook" : "http://facebook.com" },
                    { "google"   : "http://google.com" }
                ]
            };

            var output = "";

            for(var i = 0; i <= info.links.length; i++) {
                for(key in info.links[i]) {
                    if(info.links[i].hasOwnProperty(key)) {
                        output +=
                        '<li>' + 
                            '<a href="' + info.links[i][key] + '">' + key + '</a> + 
                        '</li>';
                    } // has own property check
                } // for each object
            } // for each array element

            var update = document.getElementById('links');
            update.innerHTML = output;
