What is C++

  • • General-purpose programming language
    Some Programming Paradigms C++ supports are:
      • Object-oriented
      • Functional
      • Procedural
      • Generic

    • Developed in the 1970's by Bjarne Stroustrup with the idea that it's "C with classes". Developed with functional, procedural, and generic programming in mind.
    • Used to write programs at all levels including; Applications, OS, Firmware.

  • C++ is made up of a number of different components, parts of which require understanding distinct sets of grammar. Think of it as five connected languages.
      • Preprocessor serves as the language definition.
      • C Language Syntax is the basis of C++.
      • Classes and Object are used for OOP programming.
      • Templates are used for generic programming.
      • STL (Standard Template Library) makes up the C++ Standard, and provides a tremendous amount of common functionality.

Identifiers

  • C++ identifiers are tokens that provide readable names for variables, functions, labels, and defined types. Identifiers are made up of the 26 letters of the ISO basic Latin alphabet, in lower and upper case, the 10 Wester Arabic numberals, and the ASCII underscore (_) character. They may not begin with a number, or conflict with reserved words.

  • Official standards allow identifiers to be of any length, although only the first 63 characters are guaranteed to be checked for uniqueness, and only the first 31 characters are guaranteed for external identifiers, so keep it below 31 chars.

  • An initial underscore character is commonly used for private identifiers. More than one initial underscore is normally reserved for system level private use.

Defining Variables

  • • Variables are strongly typed in C++. That means that the token representing a variable represents both its value and its type.
    • When a statement defines an integer variable and does not assign it to anything, it is said to be undefined. An undefined is a very dangerous thing, so be sure to always assign a value to a variable before you use it. Once a variable is defined, you may assign a value to it and use it. This is called initializing the variable. It's a good idea to initialize a variable as soon as possible.

  • • C++ uses qualifiers, sometimes called modifiers, that change the behavior of variables in various ways. A qualifier becomes part of the type.
    • For example, we can put the const qualifier before an int type, and this assure the value of this variable cannot change once it's defined. (Read-only variable)

Pointers

  • • Pointers are a very powerful, useful, dangerous, and common data type in C++. We first need to understand the fundamentals of how variables work however.
    • A variables is a typed and named location in memory. Memory is allocated for a value the size of the type and associated with the variable name. The variable value is copied into the memory location associated with the variable. 

                int x; //Variable definition
            x = 1; //Variable initialization
            int y = x; //Variable definition and copied initialization

  • C++ provides the ability to create a variable that is a pointer to a value as opposed to carrying the value itself. 

                int *ip; //Pointer definition
            ip = &x; //Address of ip -> Address of x
            y = *ip; //Value of y -> Value of ip

    Here we see the variable named ip is of the type pointer to int. So memory is allocated in the size of a pointer. The pointer is also strongly typed. The compiler retains an association with this pointer that it points to a value of type int.

  • Pointer Example 

                int main( int argc, char ** argv ) {
                int x = 7; 
                int y = 42;
                int ip = &x;

                printf("%d", x); //7
                printf("%d", y); //42
                printf("%d", *ip); //7

                x = 73;
                printf("%d", x); //73
                printf("%d", y); //42
                printf("%d", *ip); //73

                ip = &y;
                printf("%d", x); //73
                printf("%d", y); //42
                printf("%d", *ip); //42
            }

References

  • The C++ reference type is a lot like a pointer but with some significant differences. Let's see the first example: 

                int main( int argc, char ** argv) {
                int x = 7;
                int *ip = &x;
                int &y = x;
            }

    Here we have y as a reference and it's initialized with the address of x.

  • Remember, reference is very much like a pointer but it has some significant differences. On of the differences is that you cannot define a reference without also initializing it. When you define a reference you must say what the reference is in reference to, and in the example case, it's a reference to x. So throughout it's life, y will always be a reference to x. Anytime you use y, it's exactly the same as if you're using x.
    So if you changed the variable y, x would follow suit.

  • • References are effectively immutable, once it refers to a variable it will always refer to that same variable.
    • References are used frequently in C++, especially in functions and classes.

C-Arrays and C-Strings

  • This section is not about the STL classes called Array and String. This covers C-Arrays and C-Strings which are very different than their corresponding STL classes.

  • • An array is a fixed sizezd container for elements with a common type.
    • Arrays are indexed with integer values. The first element is always number 0. 

  •             int ia[5]; //Declaring int array of size 5
            ia[0] = 1; //Initializing the array at element 0 to 1
            *ia = 1; //Does the same exact thing as the last statement
            int *ip = ia; //Defines an int pointer and assigns the address of the array to the pointer
            *ip = 2; //Assigns 2 to the first element of the array
            ++ip; //Now points to the second element
            *ip = 3; //Assigns 3 to the second element of the array
            *(++ip) = 4; //Incrementing and using the pointer
            

  •             //Initializing an array with an initializer list
            int ia[5] = {1, 2, 3, 4, 5};

  • • A C-String is a special type of array containing characters and is terminated with a 0. Also called a null-terminated string.
    • This allows you to access the individual characters just as you would in an array.
    • Remember the 0 otherwise it will break! 

                char cString[] = {'s','t','r','i','n','g',0};
            printf("String is: %s\n", cString); //String is: string

            for(int i  = 0; cString[i]; ++i) {
                printf("Char is %c\n", cString[i]); //Prints each character in cString
            }

            //Exactly the same declaration/initialization
            char string[] = "string";
            printf("String is: %s\n", string); //String is: string

            for(char c : string) {
                if(c == 0) break;

                printf("%c", c);
            }

Conditionals

  • C++ has two basic forms of conditional statements; if else, and the ternary conditional operator. 

  •             //If-Else Example
            if(2 >= 1)
                return true;
            else if (0 <= 1)
                return true;
            else
                return false;
            

  •             //Ternary Example
            int new = 1 > 2 ? x : y;

Branching Conditional

  • C++ provides a special multi-way conditional using a switch statement. 

                //switch input must be constant
            const Type input;

            switch(input) {
                case X:
                    //Do something
                    break;
                case XX:
                    //Do something
                    break;
                case XXX:
                    //Do something
                    break;
                default:
                    //do something
                    break;
            }

While/Do Loop

  • The While loop tests the condition at the top of the loop, and can support testing conditions at the bottom.
      • Make sure to iterate out of the loop to avoid infinite loops.

  • While 

                while(condition) {
                statement
            }

  • Do/While 

                do {
                statement
            } while(condition)

For Loop

  • C++ uses an even more convenient loop structure called the for loop.

  • Structure 

                for(initialization; condition; iteration) {
                statement
            }

  • C-String For Loop 

                char s[] = "string";
            for(char * cp = s; *cp; ++cp) {
                printf("element %c\n", *cp);
            }

Range-Based Loop

  • C++ gives us a simple way of iterating over the elements of a container, using a range-based for loop.

  • Structure 

                for(Type item : container) {
                statements
            }

    Could be read as: "For each Type item in container, perform statement with that item.

  • C-String Range-Based Loop 

                char s[] = "string";

            for(char c : s) {
                if(c==0) break; //Breaks at null character
                statements
            }

Using stdout

  • In the terminology of the C++ standard library, the standard output stream is how text is displayed on the console. There's several way to send output to the standard output stream. We've seen so far, the puts and printf functions. The C-Out class is part of the standard C++ library and it works quite a bit differently than puts or printf.
      • You'll need to include <iostream> if you want to use it, as well as the std namespace.

  • Cout is a class that overloads the bitwise left shift operator to send a stream of characters to the standard output stream.
      • Used like so: 

                cout << "Hello World" << endl;
