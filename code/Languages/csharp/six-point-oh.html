Enhanced IDE (VS15)

  • Enhanced Features in Visual Studio 2015

      1. Improved syntax IntelliSense - to easily find keywords in methods and classes
      2. New lightbulb - to suggest changes
      3. Preview window - to look at modifications before committing
      4. Easier renaming/refactoring - for changing names

  • Syntax IntelliSense

    • When you hover over variables, we get color on the syntax to make it easier on the eyes when looking at keywords.
    • Outlining is displayed with rich semantics highlighting when hovering.

  • Light Bulb

    VS 2015 will now give you a lightbulb which allows you to perform certain suggested actions and will give you a preview window before executing changes.

  • Renaming and Refactoring

    You can now rename anything in real time by right-clicking the variable you want to change, and simply type it out. Other references to that thing will change as the current one does.

Expression-Level Features

  • Nameof

    You can now use the nameof reference to obtain the simple string name of a variable, type, or member.
