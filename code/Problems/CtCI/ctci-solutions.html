Arrays and Strings

  • 1.1 Unique Character Test

    Initial Questions
    • Does it need to check if it's case sensitive: abc == ABC -> false
      • Could make everything lowercase or uppercase if it doesn't matter.
    • Does it need to check if spaces count: " " == " " -> false
      • Could clean spaces from the string before checking if it doesn't matter.
    Initial Ideas
    • Create a boolean array with the size of ASCII characters (256)
    • For each character in the string, access the location of the bool array of the character parsed as an int.
    • If the value there true, return false.
    • Else, set the value there to true.
    • If it makes it through all the characters with no collision, return true.
    • Time complexity here would be O(n) where n is the length of the string, and space complexity is also O(n).
    Initial Code 
                static string UniqueChars(string str) {
                bool[] ascii = new bool[256];

                foreach(char character in str) {
                    if(ascii[(int)character])
                        return false;
                    else
                        ascii[(int)character]) = true;
                }

                return true;
            }
    Second Problem (No Data Structure)
    • NEED TO KNOW BITWISE OPERATIONS
    Additional Solutions
      1. Check every char of the string with every other char of the string for duplicate occurrences. This will take O(n^2) time and no space.
      2. If we are allowed to destory the input string, we could sort the string in O(n*log(n)) time and then linearly check the string for neighboring characters that are identical. Careful, though - many sorting algorithms take up extra space.

  • 1.2 Reverse C-String

    Initial Questions
    • What is the return case?
      • Is it returning a new string, char array, nothing (printed), itself mutated, etc.
    • Does it still need a null character?
    Some Answers
    • Manipulate original string.
    • Function Signature - 
    void reverse(char *str) {...}

    Initial Ideas
    • Create new character array and result counter to track iterating index of result
    • Loop through the string backwards.
    • Start loop right before null character.
    • Build return result from characters at loop index.
    • Add null character to end if needed. Return
    Initial Code 
                static char[] ReverseCString(char[] str) {
                char[] result = new char[str.Length];
                int resultCounter = 0;

                for(int i = str.Length-2; i >= 0; i--) {
                    result[resultCounter] = str[i];
                    resultCounter++;
                }

                return result;
            }
