Credit Card Mask

  • Usually when you buy something you're asked whether your credit card number, phone number or answer to your most secret question is still correct. However, since someone could look over your shoulder, you don't want that shown on your screen. Instead, we mask it. Task: Write a function maskify, which changes all but the last four characters into '#'.

  • Examples 

                Maskify('1234567890123'); // returns '#########0123'
            Maskify('12345'); //returns '#2345'
            Maskify('1'); //returns '1'

  • Best Solution 

                public static string Maskify(string cc) {
                int len = cc.Length;
                if(len <= 4)
                    return cc;

                return cc.Substring(len-4).PadLeft(len, '#');
            }

Number of People in the Bus

  • There is a bus moving in the city, and it takes and drop some people in each bus stop.
    You are provided a list (or array in JS) of integer arrays. Each integer array has two items which represent number of people get into bus (The first item) and number of people get off the bus (The second item).
    The second value in the first integer array is 0, since the bus is empty in the first bus stop.
    Your task is to return number of people who are still in the bus after the last bus station (after the last array).
    Please keep in mind that the test cases ensure that the number of people in the bus is always >= 0. So the return integer can't be negative.

  • Notes
    • Need to understand what data we have. There is a single List. It contains an number of integer arrays.
    • Need to loop through the list, in order to work with the data in the integer arrays.
    • Once we access the first element of each array, add that number to the final result.
    • Once we access the second element of each array, subtract that number from the final result.
    • Seems like most practical solutions involve using LINQ.

  • My Solution 

                public static int Number(List<int[]> peopleListInOut) {
                int peopleOn = 0;
                
                foreach(int[] intArray in peopleListInOut){
                    peopleOn += intArray[0];
                    peopleOn -= intArray[1];
                }
                
                return peopleOn;
            }

  • Best LINQ Solution 

                public static int Number(List peopleListInOut) { 
                return peopleListInOut.Sum(Item => Item[0] - Item[1]);
            }

  • Best Non-LINQ Solution 

                public static int Number(List peopleListInOut) { 
                int result = 0;
                
                foreach(int[] onOff in peopleListInOut) result += onOff[0] - onOff[1];
                
                return result;
            }

Sum without Highest/Lowest

  • Details

    Sum all the numbers of the array except the highest and the lowest element (the value, not the index!).

    If array is empty, null or None, or if only 1 Element exists, return 0.

                //Example

            { 6, 2, 1, 8, 10 } => 16
            { 1, 1, 11, 2, 3 } => 6

  • My Solution

                function sumArray(array) {
                
                // return if null or contains 0 or 1 items
                if(array == null || array.length <= 1)
                    return 0
                
                // take out lowest and highest elements
                array.splice(array.indexOf(Math.min.apply(null, array)), 1);
                array.splice(array.indexOf(Math.max.apply(null, array)), 1);
                
                // start summation
                var result = 0;
                for(i in array) {
                    result += array[i];
                }
                
                return result;
                }

  • Best Solution

                function sumArray(array) {
                // return if null or contains 0 or 1 items
                if (array == null || array.length < 2) {
                    return 0;
                } else {
                    // sort array lowest -> heighest
                    array = array.sort(function(a,b) {return a - b;});

                    // start summation (excluding first and last index)
                    var total = 0;
                    for (var i = 1; i < array.length - 1; i++) {
                    total += array[i];
                    }
                    return total;
                }
                }

Sum of Numbers

  • Given two integers, which can be positive and negative, find the sum of all the numbers between including them too and return it. If both numbers are equal return a or b. Note! a and b are not ordered!

  • Examples 

                GetSum(1, 0) == 1   // 1 + 0 = 1
            GetSum(1, 2) == 3   // 1 + 2 = 3
            GetSum(0, 1) == 1   // 0 + 1 = 1
            GetSum(1, 1) == 1   // 1 Since both are same
            GetSum(-1, 0) == -1 // -1 + 0 = -1
            GetSum(-1, 2) == 2  // -1 + 0 + 1 + 2 = 2

  • Best Solution 

                public int GetSum(int a, int b) {
                return (a + b) * (Math.Abs(a - b) + 1) / 2;
            }

Exes and Ohs

  • Check to see if a string has the same amount of 'x's and 'o's. The method must return a boolean and be case insensitive. The string can contains any char.

  • My Solution

                function XO(str) {
                var xCount = 0;
                var oCount = 0;
                
                for(var i in str){
                  if(str.charAt(i).toLowerCase()==="x")
                    xCount++;
                  if(str.charAt(i).toLowerCase()==='o')
                    oCount++;
                }
                
                if(xCount === 0 && oCount === 0) {
                  return true;
                } else if(xCount === oCount) {
                  return true;
                } else {
                  return false;
                }
            }

  • Best Solution

                function XO(str) {
                let x = str.match(/x/gi);
                let o = str.match(/o/gi);
                return (x && x.length) === (o && o.length);
            }

Shortest Word

  • Details

    Given a string of words, return the length of the shortest word(s).

    String will never be empty and you do not need to account for different data types.

  • My Solution

                function findShort(s) {
                // Split string into an array at spaces
                var words = s.split(' ');

                // Initialize result as the length of the first word.
                var result = words[0].length;

                // Grab shortest word through iteration
                for(var i in words) {
                    if(words[i].length <= result) {
                        result = words[i].length;
                    }
                }

                return result;
            }

  • Best Solution

                function findShort(s) {
                return Math.min.apply(null, s.split(' ').map(w => w.length));

                // Math.min.apply(null, array) - retrieve the min value of an array

                // s.split(' ') - returns an array of the string, split at the spaces

                // array.map(expression) - returns a new array with the results of the expression on every element in the calling array.

                // array.map(w => w.length) - returns a new array with the results being the length of every word in the array.
            }

Remove the Minimum

  • Details

    Given an array of integers, remove the smallest value. Do not mutate the original array/list. If there are multiple elements with the same value, remove the one with a lower index. If you get an empty array/list, return an empty array/list.
    Don't change the order of the elements that are left.

                //Examples
            removeSmallest([1,2,3,4,5]) = [2,3,4,5]
            removeSmallest([5,3,2,1,4]) = [5,3,2,4]
            removeSmallest([2,2,1,2,1]) = [2,2,2,1]

  • My Solution

                function removeSmallest(numbers) {
                numbers.splice(numbers.indexOf(Math.min.apply(null,numbers)),1);
                return numbers;

                // Math.min.apply(null, numbers) - retrieve the min value of the given array

                // numbers.indexOf(min) - returns the first index at the given element, else return -1.

                // numbers.splice(start, end) - removes a subsection from the array at the start index, until the end index.
              }

  • Best Solution

                function removeSmallest(numbers) {
                if(!numbers)return [];
                var min=Math.min.apply(null,numbers);
                numbers.splice(numbers.indexOf(min),1);
                return numbers;
              }

Number of Occurrences

                var arr = [0,1,2,2,3];
            arr.numberOfOccurrences(0) === 1;
            arr.numberOfOccurrences(4) === 0;
            arr.numberOfOccurrences(2) === 2;

  • My Solution

                Array.prototype.numberOfOccurrences = function(x) {
                
                var result = 0;
                  
                for(i in this) {
                    if(this[i]===x){
                      result++;
                    }
                }
                  
                return result;
            }

                return this.filter( function(num){ return search === num } ).length;
            }

Vampire Numbers

  • Details

                6 * 21 = 126
            # 6 and 21 would be valid 'fangs' for a vampire number as the

            10 * 11 = 110
            # 110 is not a vampire number since there are three 1's in the
            # multiplicands, but only two 1's in the product

    Create a function that can receive two 'fangs' and determine if the product of the two is a valid vampire number.

  • My Solution

                var vampire_test = function(a, b){
                let prod = a * b;
              
                let aAndB = a + "" + b;
                let test1 = aAndB.split('');
                if(test1.includes('-')){
                  test1.splice(test1.indexOf('-'), 1);
                  }
                test1 = test1.map( x => parseInt(x, 10));
                test1 = test1.sort(function(a,b) {return a-b; });

                let test2 = prodStr.split('');

                  test2.splice(test2.indexOf('-'), 1);

                  return false;

                  if(test1[i] != test2[i])
                    return false;
                  }

            }

  • Best Solution

      • You are not allowed to use any loops (including for, for/in, while, and do/while loops).

  • My Solution

                function average(scores) {
                var total = scores.reduce(function(sum, value) {
                    return sum + value;
                },0); //acummulator starts at 0

                return Math.round(total/scores.length);            
            }

Sum of nth term in Series

  • Details

    Your task is to write a function which returns the sum of following series upto nth term(parameter).

                Series: 1 + 1/4 + 1/7 + 1/10 + 1/13 + 1/16 +...

    Rules:
      • You need to round the answer to 2 decimal places and return it as String.
      • If the given value is 0 then it should return 0.00
      • You will only be given Natural Numbers as arguments.

                //Examples
            SeriesSum(1) => 1 = "1.00"
            SeriesSum(2) => 1 + 1/4 = "1.25"
            SeriesSum(5) => 1 + 1/4 + 1/7 + 1/10 + 1/13 = "1.57"

  • My Solution

                function SeriesSum(n) {
                var iterator = 1;
                var denominator = 3;
                var result = 0;
            
                for(var i = 1; iterator <= n; i+=3) {
                    result += 1/i;
                    iterator++;
                }
                return result.toFixed(2);
            }

  • Best Solution

                function SeriesSum(n) {
                for (var s = 0, i = 0; i < n; i++) {
                  s += 1 / (1 + i * 3)
                }
                
                return s.toFixed(2)
            }

Sum of Sequence

  • Details

    Your task is to make function, which returns the sum of a sequence of integers.

    The sequence is defined by 3 non-negative values: begin, end, step.

    If begin value is greater than the end, function should returns 0

                sequenceSum(2,2,2) === 2
            sequenceSum(2,6,2) === 12 // 2 + 4 + 6
            sequenceSum(1,5,1) === 15 // 1 + 2 + 3 + 4 + 5
            sequenceSum(1,5,3) === 5 // 1 + 4

  • My Solution

                const sequenceSum = (begin, end, step) => {
                var result=0;
                for(var i = begin; i <=end; i+=step) {
                  result += i;
                }
                return result;
            };

  • Best Solution

                const sequenceSum = (begin, end, step) => {
                if (begin > end) {
                  return 0;
                }
                return begin + sequenceSum(begin + step, end, step);
            };

    The Western Suburbs Croquet Club has two categories of membership, Senior and Open. They would like your help with an application form that will tell prospective members which category they will be placed.

    To be a senior, a member must be at least 55 years old and have a handicap greater than 7. In this croquet club, handicaps range from -2 to +26; the better the player the lower the handicap.

    Input:
      • Input will consist of a list of lists containing two items each. Each list contains information for a single potential member. Information consists of an integer for the person's age and an integer for the person's handicap.

                [[18, 20],[45, 2],[61, 12],[37, 6],[21, 21],[78, 9]]

    Output:
      • Output will consist of a list of string values (in Haskell: Open or Senior) stating whether the respective member is to be placed in the senior or open category.

                ["Open", "Open", "Senior", "Open", "Open", "Senior"]

  • My Solution

                function openOrSenior(data){
                var result = new Array();
                for(i in data) {
                    if(data[i][0] >= 55 && data[i][1] > 7) {
                        result.push("Senior");
                    } else {
                        result.push("Open");
                    }
                }
                return result;
            }

  • Best Solution

                function openOrSenior(data){
                function determineMembership(member){
                  return (member[0] >= 55 && member[1] > 7) ? 'Senior' : 'Open';
                }
                return data.map(determineMembership);
            }

Highest profit wins

  • Details

    Ben has a very simple idea to make some profit: he buys something and sells it again. Of course, this wouldn't give him any profit at all if he was simply to buy and sell it at the same price. Instead, he's going to buy it for the lowest possible price and sell it at the highest.

    Write a function that returns both the minimum and maximum number of the given list/array.

                minMax([1,2,3,4,5])   == [1,5]

                function minMax(arr){

            }

                return [Math.min(...arr), Math.max(...arr)];
            }

    The input to the function will be an array of three distinct numbers (Haskell: a tuple).

                gimme([2, 3, 1]) => 0
            gimme([5, 10, 14]) => 1

  • My Solution

                var gimme = function (inputArray) {
                var input = inputArray.indexOf(Math.min.apply(null, inputArray)) + inputArray.indexOf(Math.max.apply(null, inputArray));
                switch(input) {
                    case 1: {
                        return 2;
                    }
                    case 2: {
                        return 1;
                    }
                    case 3: {
                        return 0;
                    }
                }
            };

                function gimme(a) {
                return a.indexOf(a.concat().sort(function(a, b) { return a - b })[1]);
            }
