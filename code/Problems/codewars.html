Credit Card Mask

  • Usually when you buy something you're asked whether your credit card number, phone number or answer to your most secret question is still correct. However, since someone could look over your shoulder, you don't want that shown on your screen. Instead, we mask it. Task: Write a function maskify, which changes all but the last four characters into '#'.

  • Examples 

                Maskify('1234567890123'); // returns '#########0123'
            Maskify('12345'); //returns '#2345'
            Maskify('1'); //returns '1'

  • Best Solution 

                public static string Maskify(string cc) {
                int len = cc.Length;
                if(len <= 4)
                    return cc;

                return cc.Substring(len-4).PadLeft(len, '#');
            }

Sum of Numbers

  • Given two integers, which can be positive and negative, find the sum of all the numbers between including them too and return it. If both numbers are equal return a or b. Note! a and b are not ordered!

  • Examples 

                GetSum(1, 0) == 1   // 1 + 0 = 1
            GetSum(1, 2) == 3   // 1 + 2 = 3
            GetSum(0, 1) == 1   // 0 + 1 = 1
            GetSum(1, 1) == 1   // 1 Since both are same
            GetSum(-1, 0) == -1 // -1 + 0 = -1
            GetSum(-1, 2) == 2  // -1 + 0 + 1 + 2 = 2

  • Best Solution 

                public int GetSum(int a, int b) {
                return (a + b) * (Math.Abs(a - b) + 1) / 2;
            }

0-9 -> String

  • When provided with a number between 0-9, return it in words.

  • Notes
    • My solution used a simple switch statement returning each case directly.
    • Using a dictionary, you can use a key/value pair to grab from one data structure. 

  •             public static string SwitchItUp(int number) {

                var dict = new Dictionary<int, string>() {
                    {0, "Zero"},
                    {1, "One"},
                    {2, "Two"},
                    {3, "Three"},
                    {4, "Four"},

                    {6, "Six"},
                    {7, "Seven"},

                    {9, "Nine"}
                };
                //Return Value at dict Key
                return dict[number];
            }

Number of People in the Bus

  • There is a bus moving in the city, and it takes and drop some people in each bus stop.
    You are provided a list (or array in JS) of integer arrays. Each integer array has two items which represent number of people get into bus (The first item) and number of people get off the bus (The second item).
    The second value in the first integer array is 0, since the bus is empty in the first bus stop.
    Your task is to return number of people who are still in the bus after the last bus station (after the last array).
    Please keep in mind that the test cases ensure that the number of people in the bus is always >= 0. So the return integer can't be negative.

  • Notes
    • Need to understand what data we have. There is a single List. It contains an number of integer arrays.
    • Need to loop through the list, in order to work with the data in the integer arrays.
    • Once we access the first element of each array, add that number to the final result.
    • Once we access the second element of each array, subtract that number from the final result.
    • Seems like most practical solutions involve using LINQ.

  • My Solution 

                public static int Number(List<int[]> peopleListInOut) {
                int peopleOn = 0;
                
                foreach(int[] intArray in peopleListInOut){
                    peopleOn += intArray[0];
                    peopleOn -= intArray[1];
                }
                
                return peopleOn;
            }

  • Best LINQ Solution 

                public static int Number(List peopleListInOut) { 
                return peopleListInOut.Sum(Item => Item[0] - Item[1]);
            }

  • Best Non-LINQ Solution 

                public static int Number(List peopleListInOut) { 
                int result = 0;
                
                foreach(int[] onOff in peopleListInOut) result += onOff[0] - onOff[1];
                
                return result;
            }

Exes and Ohs

  • Check to see if a string has the same amount of 'x's and 'o's. The method must return a boolean and be case insensitive. The string can contains any char.

  • My Solution

                function XO(str) {
                var xCount = 0;
                var oCount = 0;
                
                for(var i in str){
                  if(str.charAt(i).toLowerCase()==="x")
                    xCount++;
                  if(str.charAt(i).toLowerCase()==='o')
                    oCount++;
                }
                
                if(xCount === 0 && oCount === 0) {
                  return true;
                } else if(xCount === oCount) {
                  return true;
                } else {
                  return false;
                }
            }

  • Best Solution

                function XO(str) {
                let x = str.match(/x/gi);
                let o = str.match(/o/gi);
                return (x && x.length) === (o && o.length);
            }
